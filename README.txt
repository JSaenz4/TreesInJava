example usage: java Cons

Will execute example usages of tree operations in java with default hardcoded example data that can be replaced in main function as is necessary. 

maxbt(tree) - find max value of binary tree.
Cons vars(Object expr) - returns set of variables in given expression.
boolean occurs(Object value, Object tree) - checks if value occurs in given tree.
Integer eval(Object tree) - evaluates given tree (an expression).
Integer eval(Object tree, Cons bindings) - evaluates given tree with binding bindings values substituted.
Cons english(Object tree) - translates expression tree into english.
String tojava(Object tree) - will translate equations in lisp form to java code.
